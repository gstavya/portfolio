# Portfolio Website

A minimalist, modern portfolio website with an aqua blue theme showcasing projects from GitHub.

## Features

- **Aqua Blue Theme**: Clean, modern design with aqua blue color scheme
- **Responsive Design**: Works seamlessly on all device sizes
- **Smooth Animations**: Scroll-triggered animations and hover effects
- **Project Showcase**: Displays all GitHub projects with descriptions and links

## Projects Included

1. **Qubit Crosstalk** - Quantum computing research project
2. **Stock Analyzer** - Python stock market analysis tool
3. **Loople** - C++ performance optimization project
4. **Earthquake Prediction** - Machine learning for seismic data
5. **ScienceBowl AI** - AI-powered Science Bowl preparation
6. **LingoLearn AI** - AI-driven language learning platform

## Getting Started

Simply open `index.html` in your web browser to view the portfolio.

## File Structure

```
portfolio_cool/
├── index.html      # Main HTML file
├── styles.css      # Styling and theme
├── script.js       # Interactive functionality
└── README.md       # This file
```

## Customization

You can easily customize:

- Colors in `styles.css` (CSS variables)
- Project information in `index.html`
- Animations and interactions in `script.js`

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
